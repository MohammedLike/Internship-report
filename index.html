Overview

Executive Summary

During this internship at Stockwiz Technologies LLP (SEBI Registered Research Analyst · INH000013925), I architected and shipped two flagship production systems that define the company's technical roadmap: the Research Report Automation System (daily ops — 2,370 stocks, 3,498 logos, <60s PDFs) and Quant Folio (strategic R&D — RAG + LLM + MCP quant platform with VectorBT backtesting and 310+ client Pine pipeline).

Highlights include a report pipeline covering 2,370 equities with 3,498 company logos, an AI Strategy Builder indexing ~1,800 strategy records across 5 Qdrant vector collections, a 212-indicator testing framework (158 TA-Lib + 54 custom SMC/pandas), and a client Pine Script pipeline processing 310+ client strategy requests. I also delivered Nifty Pharma benchmarking across 5 stocks with a 13-section PostgreSQL analysis workflow, competitive intelligence on Streak/Zerodha (209 indicators catalogued), and a working Scalping Mode prototype with live backtesting.

In-Depth Analysis

  • Problem: Each stock required 30–45 min of manual ChatGPT drafting, HTML formatting, and PDF export — unsustainable at 2,370+ instrument scale.
  • Solution: End-to-end orchestration — live market data (yfinance), templated fundamental/technical reasoning, automated React render, headless Chromium PDF via Playwright.
  • Data layer: company_profiles.csv (2,370 rows from Screener), local PNG logo library (3,498 files), NSE F&O lot-size maps.
  • Instrument support: Equity cash, index/equity futures, calls & puts with smart symbol parsing (e.g. BANKNIFTY 30 JUN 54800 PUT).
  • Business outcome: Batch generation after market close with consistent Stockwiz branding, live progress tracking, and one-click downloads.

In-Depth Analysis

  • Cognitive layer: Ollama-hosted LLMs with Qdrant RAG (BGE embeddings) for strategy hypothesis synthesis; MCP server exposes get_ohlcv, run_backtest, search_similar tools to AI agents.
  • Analytics engine: VectorBT-powered vectorized simulation with Sharpe, Sortino, CAGR, max drawdown; TA-Lib indicator catalog (112 indicators in production DB); options Greeks via Black-Scholes.
  • Data infrastructure: ~1,800 strategy records ingested (Streak, Pine scripts, indicators, backtests, OHLCV, options chain); 5 semantic Qdrant collections; Redis sub-ms backtest cache.
  • Client pipeline: 310+ client strategy requests from Excel → PostgreSQL → LLM Pine generation → human review workflow with full generation history.
  • Phase 1 companion: AI Strategy Builder (No-Code) delivers Strategy DSL parser/validator/compiler + semantic search — foundation for Phase 2 LLM synthesis.
